About the Project

This project was a dynamic web app inspired by a travel guide. It was a sort of space travel guide that also served as a way to learn about the galaxy. I wanted it to be colorful and have a theme reminiscent of the Milky Way.

This project used several tools to create a seamless user experience. These include:

  • Django
  • HTML
  • CSS/ SCSS
  • JavaScript
  • NASA Api
  • GitHub
